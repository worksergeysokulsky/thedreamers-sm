git fetch Команда git fetch связывается с удалённым репозиторием и забирает из него все изменения, которых у вас пока нет и сохраняет их локально. Мы познакомились с ней в главе Получение изменений из удалённого репозитория - Fetch и Pull и продолжили знакомство в Удалённые ветки. Мы использовали эту команду в нескольких примерах из главы Contributing to a Project. Мы использовали её для скачивания запросов на слияние (pull request) из других репозиториев в главе Pull Request Refs, также мы рассмотрели использование git fetch для работы с упакованными репозиториями в главе Создание пакетов. Мы рассмотрели тонкую настройку git fetch в главe и Спецификации ссылок.
